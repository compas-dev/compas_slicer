BaseSlicer

class compas_slicer.slicers.BaseSlicer(mesh)[source]

Bases: object

This is an organizational class that holds all the information for the slice process. Do not use this class directly in your python code. Instead use PlanarSlicer or CurvedSlicer. This class is meant to be extended for the implementation of the various slicers. See compas.slicer.slicers.PlanarSlicer and compas.slicer.slicers.CurvedSlicer as examples.

Attributes
