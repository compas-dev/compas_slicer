# Curved Slicing

Non-planar slicing using interpolation between boundary curves.

## Key Features

- `InterpolationSlicer` - Non-planar toolpaths following surface curvature
- `InterpolationSlicingPreprocessor` - Gradient field computation
- Variable layer heights based on surface geometry
- `smooth_printpoints_up_vectors()` - Tool orientation smoothing
