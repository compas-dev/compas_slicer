# Scalar Field Slicing

Slice along custom scalar field contours on a base mesh.

## Key Features

- `ScalarFieldSlicer` - Generate iso-contours from per-vertex scalar values
- Custom non-planar slicing based on distance fields
- Useful for printing on top of existing geometry
