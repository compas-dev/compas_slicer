# Vertical Sorting

Organize branching paths into vertical layers for efficient printing.

## Key Features

- `sort_into_vertical_layers()` - Group paths by connectivity
- `reorder_vertical_layers()` - Optimize print order
- Minimizes travel moves between paths
- Better for branching geometries
